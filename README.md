# Antarvis
exe file of Antarvis 2.0 Video Management System software

https://archive.org/download/antar-vis-2.0-20210317/AntarVis%202.0-20210317.exe



Central management system for ZKTeco N8000 series IPC and NVR. Supports: PTZ control and 3D positioning function, decoder and group management, 64 channels preview synchronously for each layout, 1/4/9 channel(s) playback synchronously, smart search, dual-stream playback, e-Map, Onvif.
